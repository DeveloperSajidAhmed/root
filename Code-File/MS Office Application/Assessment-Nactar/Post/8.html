হোয়াটসঅ্যাপ ভিডিও কল চলাকালীন কীভাবে আপনার স্ক্রিন শেয়ারিং করবেন August 9, 2023 by Developer Sajid WhatsApp একটি জনপ্রিয় messaging app যা ব্যবহারকারীদের text messages, photos, videos এবং voice messages পাঠাতে দেয়। সাম্প্রতিক বছরগুলিতে, WhatsApp video calls করার ক্ষমতা সহ বেশ কয়েকটি new features যুক্ত করেছে। এখন, WhatsApp আরেকটি new feature যোগ করেছে screen sharing. WhatsApp এ screen share করার জন্য, আপনাকে অ্যাপটির latest version ইনস্টল করতে হবে। আপনার latest version হয়ে গেলে, এই পদক্ষেপগুলি অনুসরণ করুনঃ WhatsApp এ স্ক্রিন শেয়ার করার সময় এখানে কিছু অতিরিক্ত জিনিস মাথায় রাখতে হবেঃ Screen sharing আপনাকে video call চলাকালীন অন্য ব্যক্তির সাথে আপনার phone screen শেয়ার করতে দেয়। এটি বিভিন্ন উদ্দেশ্যে উপযোগী হতে পারে, যেমনঃ তাদের ফোন বা কম্পিউটার নিয়ে সমস্যায় ভুগছে এমন কাউকে প্রযুক্তিগত সহায়তা দেওয়া। একদল লোকের কাছে একটি document বা presentation উপস্থাপন করা। অন্য কারো সাথে একটি project সহযোগিতা করা। WhatsApp এ screen share করার জন্য, আপনাকে অ্যাপটির latest version ইনস্টল করতে হবে। আপনার latest version হয়ে গেলে, এই পদক্ষেপগুলি অনুসরণ করুনঃ আপনি যার সাথে আপনার স্ক্রিন শেয়ার করতে চান তার সাথে একটি ভিডিও কল শুরু করুন৷ স্ক্রিনের নীচে, স্ক্রিন শেয়ারিং আইকনে আলতো চাপুন (এটি একটি arrow ফোনের মতো দেখায়)। একটি পপ-আপ উইন্ডো প্রদর্শিত হবে, আপনাকে নিশ্চিত করতে বলবে যে আপনি আপনার স্ক্রিন শেয়ার করতে চান। শেয়ার এ চাপ দিন। আপনার স্ক্রিন অন্য ব্যক্তির সাথে শেয়ার করা হবে। আপনি আপনার ফোনের চারপাশে ঘোরাফেরা করতে পারেন এবং আপনার অ্যাপগুলির সাথে স্বাভাবিকভাবে ইন্টারঅ্যাক্ট করতে পারেন। স্ক্রিন শেয়ারিং বন্ধ করতে, স্ক্রিনের নীচে শেয়ার করা বন্ধ করুন আইকনে চাপ দিন WhatsApp এ স্ক্রিন শেয়ার করার সময় এখানে কিছু অতিরিক্ত জিনিস মাথায় রাখতে হবেঃ অন্য ব্যক্তি আপনার স্ক্রিনে যা কিছু আছে তা দেখতে পারে, তাই আপনি যা শেয়ার করেন সে বিষয়ে সতর্ক থাকুন। আপনার একাধিক অ্যাপ খোলা থাকলে, অন্য ব্যক্তি শুধুমাত্র আপনি যে অ্যাপটিতে আছেন তা দেখতে পারবেন। স্ক্রিন শেয়ারিং অনেক ডেটা ব্যবহার করে, তাই নিশ্চিত করুন যে আপনি একটি Wi-Fi নেটওয়ার্কের সাথে সংযুক্ত আছেন৷ স্ক্রিন শেয়ারিং নিয়ে আপনার কোনো সমস্যা হলে, আপনার ফোন রিস্টার্ট করার চেষ্টা করুন বা হোয়াটসঅ্যাপ বন্ধ করে আবার খোলার চেষ্টা করুন। Source Link: Mark Zuckerberg https://www.facebook.com/photo?fbid=10115227233483831&set=a.612287952871 এই ব্লগ পোস্ট সহায়ক হয়েছে আশা করি!
