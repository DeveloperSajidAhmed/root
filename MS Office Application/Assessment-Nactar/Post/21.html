জাভাস্ক্রিপ্টে Non-Primitive ডেটা টাইপ July 28, 2023 by Developer Sajid জাভাস্ক্রিপ্ট হলো একটি প্রোগ্রামিং ভাষা যা বিশেষভাবে ওয়েব ডেভেলপমেন্টে ব্যবহৃত হয়। এই ভাষাতে সংখ্যা, স্ট্রিং, বুলিয়ান এবং অন্যান্য মৌলিক ডেটা টাইপ আছে, যা প্রিমিটিভ ডেটা টাইপ বলা হয়। আবার, জাভাস্ক্রিপ্টে কিছু বিশেষ ডেটা টাইপ রয়েছে যা নন-প্রিমিটিভ ডেটা টাইপ বলা হয়। এই পোস্টে, আমরা নন-প্রিমিটিভ ডেটা টাইপ এর বিভিন্ন বৈশিষ্ট্য এবং ব্যবহার নিয়ে আলোচনা করবো। নন-প্রিমিটিভ ডেটা টাইপ কি? নন-প্রিমিটিভ ডেটা টাইপ হলো একটি সাধারণ ডেটা টাইপ নয়, এটি অবজেক্ট হিসেবে পরিচিত। এই ডেটা টাইপ বিশেষভাবে অবজেক্ট, অ্যারে, ফাংশন এবং অন্যান্য কাস্টম অবজেক্ট থেকে গঠিত হতে পারে। নন-প্রিমিটিভ ডেটা টাইপ স্টোরেজ হিসেবে রেফারেন্স ব্যবহার করে, অর্থাৎ এটি ডেটা ভ্যালুর অবজেক্টের মধ্যে নেই, এবং ডেটা ভ্যালুটি ডিরেক্টলি অ্যাক্সেস করা যায় না। নন-প্রিমিটিভ ডেটা টাইপের উদাহরণ এই পোস্টে আমরা কয়েকটি উদাহরণ দিয়ে নন-প্রিমিটিভ ডেটা টাইপ সম্পর্কে আরও বুঝতে চেষ্টা করবো: অবজেক্ট: let person = { name: “জন ডো”, age: 40, isStudent: false }; অ্যারে: let fruits = [“আপেল”, “ব্যানানা”, “কাঁঠাল”]; ফাংশন: function addNumbers(a, b) { return a + b; } নন-প্রিমিটিভ ডেটা টাইপের বৈশিষ্ট্য নন-প্রিমিটিভ ডেটা টাইপের কিছু গুরুত্বপূর্ণ বৈশিষ্ট্য নিম্নলিখিত: মিউটেবলিটি: নন-প্রিমিটিভ ডেটা টাইপ মিউটেবল, অর্থাৎ আপনি একবার একটি অবজেক্ট অথবা অ্যারে তৈরি করলে আপনি এর ভ্যালু পরিবর্তন করতে পারেন। প্রোপার্টি অ্যাক্সেস: নন-প্রিমিটিভ ডেটা টাইপ এর প্রোপার্টি অ্যাক্সেস করতে আপনি ডট নোটেশন ব্যবহার করতে পারেন। console.log(person.name); // আউটপুট: “জন ডো” console.log(fruits[0]); // আউটপুট: “আপেল” মেথড কল: নন-প্রিমিটিভ ডেটা টাইপ এর মেথড কল করতে পারেন। console.log(fruits.length); // আউটপুট: 3 সমাপ্তি, এই ব্লগ পোস্টে আমরা নন-প্রিমিটিভ ডেটা টাইপ এর প্রধান বৈশিষ্ট্য এবং ব্যবহার সম্পর্কে অনেক কিছু জানতে পেরেছি। জাভাস্ক্রিপ্টে নন-প্রিমিটিভ ডেটা টাইপ ব্যবহার করা ওয়েব ডেভেলপমেন্টে অনেক উপকারী, কারণ এগুলি আমাদেরকে একাধিক মান বা তথ্য সংরক্ষণ করার সুযোগ দেয়। আমি আশা করি আপনি এই ব্লগ পোস্ট পড়ে নন-প্রিমিটিভ ডেটা টাইপ সম্পর্কে আরও ভালোভাবে জানতে পেরেছেন এবং আপনি আপনার ওয়েব ডেভেলপমেন্ট প্রকল্পে এগুলি সঠিকভাবে ব্যবহার করতে পারবেন।
