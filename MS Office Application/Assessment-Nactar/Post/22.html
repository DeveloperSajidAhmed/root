জাভাস্ক্রিপ্ট ভেরিয়েবল July 28, 2023 by Developer Sajid একটি জাভাস্ক্রিপ্ট ভেরিয়েবল কেবল স্টোরেজ অবস্থানের একটি নাম নাম অবশ্যই একটি অক্ষর (a থেকে z বা A থেকে Z), আন্ডারস্কোর (_) বা ডলার ($) চিহ্ন দিয়ে শুরু করতে হবে প্রথম অক্ষরের পরে আমরা সংখ্যা (0 থেকে 9) ব্যবহার করতে পারি, উদাহরণস্বরূপ v1 জাভাস্ক্রিপ্ট ভেরিয়েবলগুলি case sensitive, উদাহরণস্বরূপ x এবং X different ভেরিয়েবল। জাভাস্ক্রিপ্ট হলো ওয়েব ডেভেলপমেন্টের জনপ্রিয় একটি প্রোগ্রামিং ভাষা। এটি দ্বারা ওয়েব পেজের ক্লায়েন্ট সাইড স্ক্রিপ্টিং এবং ব্রাউজারে ইন্টারঅ্যাকটিভিটি প্রদান করা হয়। জাভাস্ক্রিপ্টে ভেরিয়েবল খুবই গুরুত্বপূর্ণ এবং এটি ভ্যালু সংরক্ষণের জন্য ব্যবহার করা হয়। এই ব্লগ পোস্টে আমরা জাভাস্ক্রিপ্টের ভেরিয়েবল সম্পর্কে বিস্তারিত দেখবো। ভেরিয়েবল কি? ভেরিয়েবল হলো একটি মান সংরক্ষণের জন্য একটি নাম বা আইডেন্টিফায়ার। এই ভেরিয়েবল এর মাধ্যমে আমরা ভ্যালু স্টোর করতে পারি এবং সেই ভ্যালু কে পরবর্তীতে ব্যবহার করতে পারি। জাভাস্ক্রিপ্টে ভেরিয়েবল বিভিন্ন ডাটা টাইপ স্টোর করতে পারে, যেমন সংখ্যা, স্ট্রিং, বুলিয়ান, অবজেক্ট, অ্যারে ইত্যাদি। জাভাস্ক্রিপ্টে ভেরিয়েবল ডিক্লেয়ার করা জাভাস্ক্রিপ্টে ভেরিয়েবল ডিক্লেয়ার করার জন্য var, let এবং const এই তিনটি কীওয়ার্ড ব্যবহার করা হয়। তাহলে আমরা এই তিনটি ডিক্লেয়ারেশন প্রক্রিয়া বিশেষভাবে দেখি: // var ব্যবহার করে ভেরিয়েবল ডিক্লেয়ার করা var name = “আলেকস”; // let ব্যবহার করে ভেরিয়েবল ডিক্লেয়ার করা let age = 30; // const ব্যবহার করে ভেরিয়েবল ডিক্লেয়ার করা const isStudent = true; উপরে, var, let, এবং const এর মাধ্যমে আমরা একই নামের ভেরিয়েবল তিনটি বিভিন্ন ভ্যালু স্টোর করতে পেরেছি। ভেরিয়েবলে ভ্যালু পরিবর্তন করা জাভাস্ক্রিপ্টে ভেরিয়েবলে ভ্যালু পরিবর্তন করা খুবই সহজ। আপনি কোনও সময়ে ভেরিয়েবলের মান পরিবর্তন করতে পারেন এবং সেই মান পরবর্তীতে ব্যবহার করতে পারেন: let count = 0; console.log(count); // আউটপুট: 0 count = 10; console.log(count); // আউটপুট: 10 count = count + 5; console.log(count); // আউটপুট: 15 উপরে, আমরা প্রথমে count ভেরিয়েবল স্টোর করেছি 0। তারপরে আমরা এটির মান পরিবর্তন করে 10 এ সেট করেছি, এবং শেষে আমরা একটি ভ্যালু যোগ করে 15 এ সেট করেছি। Ex. var x = 10; var y = 20; var z = x+y; document.write(z); variable – চলক x – data data name দিলাম – x x – variable 10 – value Var হলো variable declare করার জন্য। (;) সেমিক্লোন দেওয়া বাধ্যতামূলক না।statement শেষ করতে দেই। সমাপ্তি, জাভাস্ক্রিপ্টে ভেরিয়েবল একটি মৌলিক সংজ্ঞায়িত বিষয়। এটি আপনাকে ভ্যালু স্টোর করতে এবং পরবর্তীতে ব্যবহার করতে সাহায্য করে। আমরা দেখেছি কিভাবে জাভাস্ক্রিপ্টে ভেরিয়েবল ডিক্লেয়ার করা এবং ভেলু পরিবর্তন করা হয়। আপনি এই সম্পর্কে আরও বেশি জানতে চাইলে, জাভাস্ক্রিপ্ট ডকুমেন্টেশন পড়তে সময় ব্যয় করতে পারেন। আমরা আশা করি আপনি এখান থেকে জাভাস্ক্রিপ্টের ভেরিয়েবল এর মূল ধারণা সম্পর্কে ভালো ধারণা পেয়েছেন। ভবিষ্যতে আপনি জাভাস্ক্রিপ্টে ভেরিয়েবল এর বিভিন্ন ব্যবহার সম্পর্কে আরও গভীর অধ্যয়ন করতে সাহায্য পেতে পারেন!
