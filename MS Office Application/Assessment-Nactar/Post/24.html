ব্রাউজার এবং Node.js-এ JavaScript কোড চালানো July 28, 2023 by Developer Sajid জাভাস্ক্রিপ্ট উদাহরণঃ কোড করা সহজ জাভাস্ক্রিপ্ট কোড ৩ ভাবে লেখা যায়ঃ বডি ট্যাগের মধ্যে (write body before closing tag best practice use script tag) হেড ট্যাগের মধ্যে external জাভাস্ক্রিপ্ট ফাইল। ওয়েব ডেভেলপমেন্টে প্রাথমিক ধাপ হিসেবে জাভাস্ক্রিপ্ট এর পরিচিতি খুব গুরুত্বপূর্ণ। জাভাস্ক্রিপ্ট হলো একটি স্ক্রিপ্টিং ভাষা যা ওয়েব পেজে ইন্টারেক্টিভ কম্পোনেন্ট সৃষ্টি করার জন্য ব্যবহৃত হয়। আমরা এই পোস্টে দেখবো কিভাবে সহজভাবে একটি জাভাস্ক্রিপ্ট কোড লেখা এবং তা রান করতে পারি। আসুন শুরু করা যাক! ধাপ ১: টেক্সট এডিটর বেছে নিন প্রথমে, আপনি একটি টেক্সট এডিটর বেছে নিতে হবে। টেক্সট এডিটর আপনার কোড লেখার জন্য প্রয়োজনীয়। আপনি ব্যক্তিগত কাজের জন্য Notepad ব্যবহার করতে পারেন বা এডভান্সড এডিটর যেমন Visual Studio Code, Sublime Text বা Atom ব্যবহার করতে পারেন। এই পোস্টে আমরা Visual Studio Code ব্যবহার করবো। এটি বিনামূল্যে সাধারণ এবং বিশেষজ্ঞদের মধ্যে খুব জনপ্রিয়। ধাপ ২: কোড লেখা এখন, আমরা নিচের মতো জাভাস্ক্রিপ্ট কোড লেখতে পারি: console.log("হ্যালো জাভাস্ক্রিপ্ট!"); উপরের কোডে, console.log() ফাংশন ব্যবহার করে আমরা “হ্যালো জাভাস্ক্রিপ্ট!” প্রিন্ট করতে পারি। ধাপ ৩: ফাইল সংরক্ষণ এবং নাম প্রয়োগ এখন, আপনি ফাইলটি সংরক্ষণ করুন এবং এটির নাম দিন “hello.js”। এই নাম দেওয়ার মাধ্যমে এটি একটি জাভাস্ক্রিপ্ট ফাইল হিসেবে চিহ্নিত হয়। ধাপ ৪: কোড রান করা ফাইলটি সংরক্ষণ করার পরে, আপনি আপনার কোড রান করতে পারেন। জনপ্রিয় ওয়েব ব্রাউজার Google Chrome বা Mozilla Firefox ব্যবহার করুন। এটি পারফর্মেন্স উপর কিছু মিল রয়েছে তাহলে আপনি Node.js ব্যবহার করতে পারেন যেটি জাভাস্ক্রিপ্ট এর সার্ভার সাইড রানটাইম এনভায়রনমেন্ট প্রদান করে। আপনি ব্রাউজারে নতুন ট্যাবে গিয়ে ফাইলটি খোলতে পারেন, বা যদি আপনি Visual Studio Code ব্যবহার করেন, সেখান থেকে “Run” মেনু ব্যবহার করে ফাইলটি রান করুন। ধাপ ৫: প্রদর্শন দেখা এখন আপনি আপনার কোড সঠিকভাবে রান করেন। আপনি আপনার ব্রাউজারে কনসোল দেখতে পাবেন “হ্যালো জাভাস্ক্রিপ্ট!” মেসেজ। document.write() function is userd to display dynamic content through JavaScript. NodeJS দিয়ে রান করা যায় JavaScript ES6 দিয়ে রান করা যায় Property – document Method – write document.write(“Hello JS”) সমাপ্তি, এই পোস্টে, আমরা দেখেছি কিভাবে সহজভাবে জাভাস্ক্রিপ্টে একটি প্রোগ্রাম লেখা এবং এটি রান করতে পারি। আপনি এবং আপনার পরিবারের সদস্যরা এই প্রক্রিয়াটি ব্যবহার করে শুরু করতে পারেন ওয়েব ডেভেলপমেন্ট প্রকল্প এবং আরও অনেক কিছু শেখার সুযোগ পেতে পারেন। তাহলে শুভ কামনা করি আপনার সফলতা এবং প্রগতির জন্য!
