জাভাস্ক্রিপ্ট শেখার আগে আপনার যা জানা দরকার July 28, 2023 by Developer Sajid JavaScript শেখার জন্য আমাদের প্রয়োজন Code Editor (JavaScript Code লেখার জন্য) Browser (JavaScript Code এর আউটপুট দেখার জন্য) আমাদের কম্পিউটার অথবা লেপটপ এ Code Editor এবং Browser থাকলেই আমরা JavaScript শেখা শুরু করতে পারি। Code Editor জনপ্রিয় কোড ইডিটরগুলি হলোঃ Visual Studio Code, Sublime Text, Bracket, Notepad++ অথবা Atom। আপ্নাদের যদি পছন্দের কোন কোড ইডিটর না থাকে তাহলে আমি বলবো Visual Studio Code দিয়ে শুরু করুন। আমি নিজে Visual Studio Code ব্যবহার করি। Browser জনপ্রিয় ওয়েব ব্রাউজারগুলি হলোঃ Google Chrome, Firefox অথবা Microsoft Edge। আপ্নাদের যদি পছন্দের কোন ওয়েব ব্রাউজার না থাকে তাহলে আমি বলবো Google Chrome দিয়ে শুরু করুন। আমি নিজে Google Chrome ব্যবহার করি। জাভাস্ক্রিপ্ট বর্তমানে ওয়েব ডেভেলপমেন্টের জনপ্রিয় একটি মূল্যবান প্রোগ্রামিং ভাষা। আপনি শুরু করতে পারেন উপরে উল্লিখিত উপায়ে জাভাস্ক্রিপ্ট শেখার প্রক্রিয়াকে সম্পন্ন করতে। স্বল্প সময় পরে আপনি ওয়েব ডেভেলপমেন্ট স্কিলস এবং প্রকৌশল প্রয়োজন অনুভব করবেন, যা আপনার ওয়েব ডেভেলপমেন্ট ক্যারিয়ারকে এগিয়ে নিতে সাহায্য করবে।
