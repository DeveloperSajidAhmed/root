HTML কোড কিভাবে রান করতে হয় July 28, 2023 by Developer Sajid ওয়েব ডেভেলপমেন্টে শুরু করতে গিয়ে আপনি প্রথমে হাতে কলমে HTML কোড বেশিরভাগ সময় লেখতে থাকবেন। ওয়েব পেজ বা ওয়েব এপ্লিকেশনের স্থানীয়ভাবে দেখতে এবং পরীক্ষা করতে আপনি আপনার কম্পিউটারের ব্রাউজারে HTML কোড রান করতে পারেন। এই পোস্টে, আমরা এই প্রক্রিয়ার সাথে সাথে পরিচিত হব। প্রথমেই, আমরা যে কিছু জিনিস প্রয়োজন থাকবে তা সমীক্ষা করা যাক: ১. টেক্সট এডিটর: একটি টেক্সট এডিটর আপনার কোড লেখার জন্য প্রয়োজন। টেক্সট এডিটর হতে পারে Notepad অথবা Notepad++, Sublime Text, Visual Studio Code, ইত্যাদি। এই প্রস্তুতি অনুযায়ী, আমি Visual Studio Code ব্যবহার করবো। এটি বিনামূল্যে সাধারণ এবং বিশেষজ্ঞদের মধ্যে খুব জনপ্রিয়। ২. ব্রাউজার: একটি ব্রাউজার প্রয়োজন যা আপনার ওয়েব পেজ বা ওয়েব এপ্লিকেশনটি দেখার জন্য ব্যবহার করতে পারেন। মূলত, Google Chrome, Mozilla Firefox, এবং Microsoft Edge পপুলার ওয়েব ব্রাউজার। এখন, নিচে দেওয়া ধাপগুলি অনুসরণ করে দেখা যাক কিভাবে HTML কোড রান করতে হয়: ধাপ ১: টেক্সট এডিটরে কোড লেখা প্রথমে, টেক্সট এডিটর খুলুন এবং একটি নতুন ফাইল সংযুক্ত করুন। তারপরে এই ফাইলের ভেতরে নীচের মতো HTML কোড লেখুন: আমার প্রথম ওয়েব পেজ

হ্যালো, এটি আমার প্রথম ওয়েব পেজ!

এটি আমি ভাষায় লিখেছি।

ধাপ ২: ফাইল সংরক্ষণ এবং নাম প্রয়োগ এখন আপনি ফাইল সংরক্ষণ করুন এবং এটির নাম দিন “index.html”। এই নাম দেওয়ার মাধ্যমে এটি একটি প্রধান ওয়েব পেজ হিসেবে চিহ্নিত হয়। ধাপ ৩: ব্রাউজারে ওপেন করা ফাইলটি সংরক্ষণ করার পরে, আপনি এটি একটি ব্রাউজারে দেখতে পাবেন। ব্রাউজারে ফাইল উপরে ড্যাবল-ক্লিক করুন বা যদি আপনি Visual Studio Code ব্যবহার করেন, তাহলেOpen with Live Server Extension ব্যবহার করেও দেখতে পাবেন। যদি আপনি ব্রাউজার নির্ধারিত না করেন তবে আপনি ব্রাউজারে নতুন ট্যাবে গিয়ে ফাইলটি খোলতে পারেন, কিংবা ফাইলটির পথ বারে “file://পথ/অবশ্যই-একটি-ইনডেক্স-ফাইল.html” টাইপ করে সরাসরি ব্রাউজারে ইনপুট করুন। ধাপ ৪: প্রদর্শন দেখা আপনি এখন সঠিকভাবে তারিখ, মুদ্রণ হওয়া ওয়েব পেজ দেখতে পান। যদি আপনি ধাপের মধ্যে কোনও ত্রুটি পেতে থাকেন, তবে আপনার লেখা সঠিক হয়েছে কিনা পরীক্ষা করুন। এখানে, আপনি শেখা এবং চেষ্টা করা হলো কিভাবে আপনি স্থানীয়ভাবে আপনার কম্পিউটারে একটি সাধারণ HTML পেজ রান করতে পারেন। আপনি এবং আপনার পরিবারের সদস্যরা এই প্রক্রিয়াটি ব্যবহার করে শুরু করতে পারেন ওয়েব ডেভেলপমেন্ট প্রকল্প এবং আরও অনেক কিছু শেখার সুযোগ পেতে পারেন। তাহলে শুভ কামনা করি আপনার সফলতা এবং প্রগতির জন্য!
